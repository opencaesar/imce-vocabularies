  • metre-per-second kilometre-per-second_of_metre-factor kilometre

4. Aspects

4.1. UnitOfLength

Supertypes: metrology:CoherentMeasurementUnit

Subtypes: kilometre, metre

Properties with domain: metrology:isDimensionlessMeasurementUnit (must have value false)

4.2. UnitOfAngularMeasure

Supertypes: metrology:CoherentMeasurementUnit

Subtypes: radian

Properties with domain: metrology:isDimensionlessMeasurementUnit (must have value false)

4.3. UnitOfTime

Supertypes: metrology:CoherentMeasurementUnit

Subtypes: millisecond, second

Properties with domain: metrology:isDimensionlessMeasurementUnit (must have value false)

4.4. UnitOfVelocity

Supertypes: metrology:CoherentMeasurementUnit

Subtypes: kilometre-per-second, metre-per-second

Properties with domain: metrology:isDimensionlessMeasurementUnit (must have value false)

4.5. UnitOfRotation

Supertypes: metrology:CoherentMeasurementUnit

Subtypes: revolutions

Properties with domain: metrology:isDimensionlessMeasurementUnit (must have value false)

4.6. UnitOfFrequency

Supertypes: metrology:CoherentMeasurementUnit

Subtypes: hertz

Properties with domain: metrology:isDimensionlessMeasurementUnit (must have value false)
